# one-hand-keyboard-AHK-scripts
Overview
========

This is my personal keyboard layout. It is made for the TECK (“Truly Ergonomic
Keyboard”). There’s also a port for the Ergodox EZ. I hope it may be of
inspiration to someone.
```
Layer 1-----------------------------------------

|     |        | TAB | ENT |  X    |        |
|     |   Q  |   H    |   T    |   I    |   P   |
|     |   S  |   E     |   ^     [  A   ]  W   |
|     |   N  |    L    |  BS   |   D  |   K   |
```
```
Layer 2-----------------------------------------

|       |        |        |  BS  |        |       |
|       |   Z   |  B   | ENT |  G   |   J  |
|       |   U   |  O  |    ^    [  R   ]  C   |
|       |  M   |  Y  |    V    |  F   |   P  |
```
```
Numpad 3

|     |        |         |  BS  |        |        |
|     |        |    7   |   8   |   9   |        |
|     |        |    4   |   5   [   6   ]   0   |
|     |        |    1   |   2   |   3   |        |
```
